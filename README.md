# GridGeneration

## Documentation

See the documentation and development for [GridGeneration.jl](https://marvyn.com/GridGeneration.jl/dev/)

## Setting Up

Add the package to your project:

```julia
using Pkg
Pkg.add("GridGeneration")
```

and include in your script:

```julia
using GridGeneration
```
